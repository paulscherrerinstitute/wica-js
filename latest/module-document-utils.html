Module: document-utils

Provides helper functions for wica-aware html pages.

(inner) findWicaElements() → {NodeListOf.<Element>}

Finds all "wica-aware" HTML elements in the current document. That's to say, all elements which include an attribute which defines the name of the wica channel.
